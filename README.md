# ChargeGuard ⚡

An Android app that alerts you instantly when your charger stops working.

## Problem it solves
When a charger is connected but stops delivering power (loose cable, faulty
adapter, power cut), your phone silently drains. ChargeGuard detects this
instantly and alerts you.

## Features
- Real-time battery %, voltage and temperature monitoring
- Instant notification + vibration when charger disconnects
- Persistent event history
- Stop alert from notification or in-app button
- Works while app is in background

## Tech Stack
- Kotlin
- BroadcastReceiver (dynamic)
- NotificationManager
- SharedPreferences
- ViewBinding
- Material Design
